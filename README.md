# Python-Practical-Works
Collection complète de travaux pratiques Python couvrant la programmation orientée objet : classes, encapsulation, héritage simple et multiple, polymorphisme. Exercices corrigés avec comptes bancaires, vecteurs 2D, système scolaire, véhicules connectés et gestion de bibliothèque. Idéal pour maîtriser les concepts fondamentaux de la POO Python
