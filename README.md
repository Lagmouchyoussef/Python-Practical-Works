## 7. Exercises

### Exercise 1: Bank Account System 🏦

**Difficulty**: ⭐ Beginner | **Focus**: Encapsulation

```mermaid
classDiagram
    class BankAccount {
        +holder: str
        -_bank: str
        --balance: float
        +__init__(holder, initial_balance)
        +deposit(amount): bool
        +withdraw(amount): bool
        +get_balance(): float
        +__str__(): str
    }
```

**Key Concepts**:
- `__init__()` - Constructor
- `self` - Instance reference
- `__balance` - Private attribute
- `get_balance()` - Getter method

### Exercise 4: Connected Vehicles 🚗

**Difficulty**: ⭐⭐⭐ Advanced | **Focus**: Multiple Inheritance

```mermaid
classDiagram
    class Vehicle {
        +brand: str
        +max_speed: int
        +__init__(brand, max_speed, **kwargs)
        +description()
    }
    
    class ElectricVehicle {
        +range_km: int
        +__init__(brand, max_speed, range_km, **kwargs)
        +description()
    }
    
    class ConnectedVehicle {
        +os_system: str
        +__init__(brand, max_speed, os_system, **kwargs)
        +description()
    }
    
    class ConnectedElectricCar {
        +__init__(brand, max_speed, range_km, os_system)
        +description()
    }
    
    Vehicle <|-- ElectricVehicle
    Vehicle <|-- ConnectedVehicle
    ElectricVehicle <|-- ConnectedElectricCar
    ConnectedVehicle <|-- ConnectedElectricCar
```

### Final Project: Library System 📚

**Difficulty**: ⭐⭐⭐⭐ Expert | **Focus**: Complete Project

```mermaid
classDiagram
    class Document {
        +title: str
        +author: str
        +year: int
        +__init__(title, author, year)
        +display()
        +__str__()
        +__repr__()
    }
    
    class Book {
        +num_pages: int
        +isbn: str
        +__init__(title, author, year, num_pages, isbn)
        +display()
        +__str__()
        +__eq__()
    }
    
    class ScientificArticle {
        +journal: str
        +doi: str
        +__init__(title, author, year, journal, doi)
        +display()
    }
    
    class Library {
        +name: str
        +documents: list
        +__init__(name)
        +add(document): bool
        +search(title): Document
        +display_catalog()
        +__len__()
        +__contains__()
    }
    
    Document <|-- Book
    Document <|-- ScientificArticle
    Library --> "*" Document
```
